# Package index

## Main function

The main generic function

- [`eikos()`](https://rwoldford.github.io/eikosograms/reference/eikos.md)
  : Generic method for creating an eikosogram

## S3 methods

Implemented methods

- [`eikos(`*`<default>`*`)`](https://rwoldford.github.io/eikosograms/reference/eikos.default.md)
  : Create a new eikosogram
- [`eikos(`*`<formula>`*`)`](https://rwoldford.github.io/eikosograms/reference/eikos.formula.md)
  : Draw eikosogram using a formula to identify response and
  conditioning variates

## Helper functions

Functions not generally called by user, but by eikos

- [`eikos_data()`](https://rwoldford.github.io/eikosograms/reference/eikos_data.md)
  : Create eikosogram data frame
- [`eikos_legend()`](https://rwoldford.github.io/eikosograms/reference/eikos_legend.md)
  : Create eikosogram legend
- [`eikos_x_labels()`](https://rwoldford.github.io/eikosograms/reference/eikos_x_labels.md)
  : eikos helper function. Returns grob with x axis labels.
- [`eikos_x_probs()`](https://rwoldford.github.io/eikosograms/reference/eikos_x_probs.md)
  : Create grob with eikosogram x-axis probabilities
- [`eikos_y_labels()`](https://rwoldford.github.io/eikosograms/reference/eikos_y_labels.md)
  : eikos helper function. Returns grob with y axis labels.
- [`eikos_y_probs()`](https://rwoldford.github.io/eikosograms/reference/eikos_y_probs.md)
  : Create grob with eikosogram y-axis probabilities
